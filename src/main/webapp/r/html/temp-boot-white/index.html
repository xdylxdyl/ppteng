杀人游戏是一类智力和心力游戏。

概括地说，是一个多人参与的较量口才和分析判断能力（推理）的游戏，当然，心理素质在中间也起着很关键的作用。

游戏通常分为两大阵营，好人方和杀手方；好人方以投票为手段投死杀手获取最后胜利，杀手方隐匿于好人中间，靠夜晚杀人及投票消灭好人方成员为获胜手段。
